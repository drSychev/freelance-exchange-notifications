## Installation

```bash
$ npm install
```
```bash
$ cp .env.example .env
```
Заполнить переменные в файле .env значениями из настроек плагина


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
